⚠️ Power Query Action Required

The House dataset has no column headers. In Power Query Editor: select "Use First Row as Headers" is NOT applicable here. Instead, manually rename columns to: CRIM, ZN, INDUS, CHAS, NOX, RM, AGE, DIS, RAD, TAX, PTRATIO, B, LSTAT, MEDV. The target variable MEDV = Median value of owner-occupied homes (×$1,000).

DAX Measures — Churn Dataset
// Copy-paste ready for Power BI Desktop
MEASURE 1 · Churn Rate
Churn Rate = 
DIVIDE(
  COUNTROWS(FILTER(Churn, Churn[Churn] = "True")),
  COUNTROWS(Churn),
  0
)
MEASURE 2 · Avg Day Charge (Churned)
Avg Day Charge Churned = 
CALCULATE(
  AVERAGE(Churn[Total day charge]),
  Churn[Churn] = "True"
)
MEASURE 3 · Intl Plan Churn Rate
Intl Churn Rate = 
CALCULATE(
  [Churn Rate],
  Churn[International plan] = "Yes"
)
MEASURE 4 · RANKX by State Churn
State Churn Rank = 
RANKX(
  ALL(Churn[State]),
  [Churn Rate],
  ,DESC,Dense
)

Positive Pct =

Posts YTD =

Sentiment Group =
